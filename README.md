# Dynamic Array Operations in C

This repository contains basic C programs that demonstrate common array operations using user-defined input and functions. These programs are designed to help beginners understand array manipulation, function usage, and simple problem-solving techniques in C.

## Programs Included

- **user-input-size.c** – Accepts the array size and elements from the user.
- **max.c** – Finds the maximum element in an array.
- **function-arrays.c** – Demonstrates passing arrays to functions.
- **odd-even-sum.c** – Calculates the sum of even and odd elements separately.
- **pair.c** – Identifies and displays pairs of elements based on the program logic.

## Concepts Covered

- User-defined array size
- Array input and output
- Functions with arrays
- Maximum element search
- Even and odd number processing
- Pair identification
- Basic array traversal

## Technologies Used

- Language: C
- Compiler: GCC / Turbo C / Code::Blocks / VS Code

## Learning Objective

The purpose of this repository is to strengthen the fundamentals of arrays in C by implementing commonly used array operations and function-based programming techniques.
